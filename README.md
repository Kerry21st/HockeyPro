## О проекте

HockeyPro — это современный интернет-магазин, специализирующийся на продаже профессиональной и любительской хоккейной экипировки. Сайт предлагает удобный интерфейс для выбора товаров, подбора экипировки по параметрам пользователя, а также возможность оставлять отзывы и управлять личным кабинетом.

## Функциональность

- **Каталог товаров** — карточки товаров с фотографиями, описанием, ценой и кнопкой добавления в корзину
- **Система лайков** — возможность добавлять товары в избранное
- **Корзина** — управление количеством товаров, расчет итоговой суммы, применение промокода
- **Личный кабинет** — просмотр и редактирование профиля, история заказов
- **Авторизация и регистрация** — вход через email и социальные сети
- **Подбор экипировки** — форма с параметрами для рекомендации товаров
- **Страница товара** — подробное описание, характеристики, видеообзоры, отзывы с возможностью добавления фото
- **Отзывы** — система рейтинга и отзывов с загрузкой фотографий
- **Слайдер с брендами** — карусель с изображениями команд-партнеров
- **Адаптивный дизайн** — корректное отображение на мобильных устройствах, планшетах и ПК
- **Бургер-меню** — навигация для мобильных устройств


## Используемые технологии

- **HTML5** — семантическая верстка
- **CSS3** — Flexbox, Grid, анимации, медиазапросы, CSS переменные
- **JavaScript** — работа с DOM, обработка событий, модальные окна, переключение вкладок
- **Swiper.js** — карусель для слайдера с брендами
- **Яндекс.Карты** — интеграция карты на странице контактов
